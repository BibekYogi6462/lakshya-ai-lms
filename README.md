🎯 LAKSHYA – AI Integrated LMS

LAKSHYA is a modern AI-powered Learning Management System (LMS) built using the MERN stack (MongoDB, Express.js, React.js, Node.js).
It provides a seamless, intelligent, and interactive learning platform for both students and instructors, leveraging AI-driven features and scalable web technologies.

🚀 Key Features
🧠 AI-Powered Smart Search (Gemini Integration)
Discover courses and content instantly with intelligent AI-based recommendations.
🔐 Google Authentication
Secure and hassle-free login/signup using Google OAuth.
💳 Payment Gateway Integration
Reliable and secure system for purchasing courses.
⚛️ Redux Toolkit State Management
Scalable and efficient global state handling.
📚 Student & Instructor Dashboards
Dedicated dashboards for managing courses, tracking progress, and analytics.
🖥️ Fully Responsive Design
Optimized for all screen sizes — desktop, tablet, and mobile.

⚙️ Installation & Setup
1. Clone the Repository
git clone https://github.com/your-username/LAKSHYA-AI-LMS.git
cd LAKSHYA-AI-LMS
2. Setup Backend
cd backend
npm install
npm start
3. Setup Frontend
cd frontend
npm install
npm start
🔑 Environment Variables

Create a .env file in the backend directory:

MONGO_URI=your_mongodb_url
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret
RAZORPAY_KEY=your_razorpay_key
GEMINI_API_KEY=your_gemini_api_key
